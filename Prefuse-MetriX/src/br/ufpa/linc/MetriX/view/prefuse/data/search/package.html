Text search support for prefix, keyword, and regular expression searches. Each search engine type subclasses {@link br.ufpa.linc.MetriX.view.prefuse.data.search.SearchTupleSet}, an implementation of the {@link br.ufpa.linc.MetriX.view.prefuse.data.tuple.TupleSet} interface. Classes derived from SearchTupleSet (e.g., {@link br.ufpa.linc.MetriX.view.prefuse.data.search.PrefixSearchTupleSet}, {@link br.ufpa.linc.MetriX.view.prefuse.data.search.KeywordSearchTupleSet}, and {@link br.ufpa.linc.MetriX.view.prefuse.data.search.RegexSearchTupleSet}) can be instantiated and used directly or in conjunction with a {@link br.ufpa.linc.MetriX.view.prefuse.data.query.SearchQueryBinding}.
